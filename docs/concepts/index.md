## Overview

alpha-python is structured around a few key architectural patterns:

- **Repository Pattern** — abstracts data access behind a consistent interface
- **Unit of Work** — groups repository operations into transactional units
- **Dependency Injection** — wires components together via YAML configuration
- **Factory Pattern** — handles complex object creation
